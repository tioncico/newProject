## admin_list 后台用户列表
- adminId
- adminName         ```VARCHAR(30) NOT_NULL 用户名 DEFAULT='默认用户'```
- adminAccount      ```VARCHAR(20) NOT_NULL 账号```
- adminPassword     ```VARCHAR(32) NOT_NULL 密码``` 
- adminSession      ```VARCHAR(32) 会话信息``` 
- lastLoginIp       ```VARCHAR(20) 最后一次登录ip``` 
- lastLoginTime     ```INT(10) 最后一次登录时间``` 
- addTime           ```INT(10) NOT_NULL 用户添加时间```  
- isForbid          ```tinyint (2) 是否禁用 0```
- isDelete          ```tinyint(2) 是否删除 0```

## article_category_list 文章分类列表
- categoryId `int 10 分类id`
- categoryName `varchar 64 分类名称`
- note `varchar 255 该分类备注`
- pid `int 10 父级分类id`

## article_list 文章列表
- articleId `int 10 文章id`
- categoryId `int 10 分类id`
- categoryName `varchar 64 分类名称`
- title `varchar 64 标题 `
- imgUrl `varchar 255 图片地址`
- description `varchar 255 简介`
- author `varchar 32 作者`
- content `text 内容`
- addTime `int 10 新增时间`
- updateTime `int 10 更新时间`
- note `varchar 255 文章备注`
- state `tinyint 1正常,2隐藏`

## web_setting_list 网站配置表
- id `int`
- name  `varchar 64 配置名`
- note `varchar 255 备注`
- value `text 配置值`
- type `varchar 32 数据类型 1文本,2编辑框,3编辑器,4图片,5文件`
